Project Stage 0: Getting Started with HTML

The Basics of the Web and HTML

World Wide Web – collection of HTML docs (Hyper Text Markup Language)
Hyper Links link pages

Types of files that can exist on web –
HTML, plain text, images, video, music, etc.

Major Pieces of Web:
- You
- Computer running browser (program to display files found on web)
- Internet (protocol HTTP to communicate with servers)
- Servers (computers that host files)

HTML made up of:
- Text content
- Markup
- References to other documents
- Links to other pages

HTML Markup – made up of tags:
Contents (opening tag, contents, closing tag – whole thing is an element)
B tag: makes contents bold - inline element
Em tag: makes contents italics - inline element
If you forget closing tag, everything after opening tag has that characteristic

HTML Attributes:
contents
Anchor tag - For making links "<"a href=”www.w3c.org/html”>contents"<"/a>
Image tag - For adding images "<"img src=”url of image to download” alt=”text to display if image doesn’t display”>
Need text in case image doesn’t load; no closing tag needed

Whitespace
In HTML, all whitespace (spaces, returns) appears as a single space
Break tag - <"br"> inline element; creates a line break when rendering text (no closing tag – “void” tag)
Paragraph tag - <"p"> (needs closing tag) block element; makes an invisible box around the text

Other Elements:
Span/div elements
Span – inline "<"span>"<"/span>
Div – block "<"div>"<"/div>
